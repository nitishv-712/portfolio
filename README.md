# 3D Portfolio Room

Highly detailed React Three Fiber room with:
- PBR textures (oak wood, brick, rug)
- Realistic lighting (Environment HDR, contact shadows, area lights)
- Animations (Float, smooth OrbitControls)
- Interactive books, theme toggle, UI overlays

## Run
```bash
pnpm dev
```
Open http://localhost:3000

Enhanced by BLACKBOXAI for photoreal visuals.
